CityEvents

Brief description of the developed system

CityEvents is a startup project whose goal is to allow users to quickly post and find events in all cities of Russia.

Generalized functional requirements:
  • Admin Panel
  • Content Management
  • My account
  • Search system for events and events
  • Event and Event Posting System
  • Hosted event management system
  • Displaying an event on a map
  • Differentiation of user levels
Technology Stack:
  • HTML
  • CSS
  • JQuery
  • PHP
  • REST API
  • JWT
  • MySQL
