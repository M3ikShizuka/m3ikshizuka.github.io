ShizukaFramework

Brief description of the developed system

ShizukaFramework is a graphical framework for developing GUI applications. Support DirectX, OpenGL, Engine render (Hook software graphics interface).

GUI elements:
  • Page
  • View
  • Button
  • Check box
  • Input field
  • Text field
  • Drop-down menu
  • Scrollbar
  • Image
Technology stack:
  • C++
  • DirectX
  • OpenGL
Render:
  • OpenGL 1.2, 3.3, 4.6
  • DirectX 9, 11
  • Engine render (Hook software graphics interface)
Platforms:
  • Windows
  • Linux (port in plans)
  • macOS (port in plans)
