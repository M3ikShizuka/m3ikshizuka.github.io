EtenDelivery

Brief description of the developed system

EtenDelivery - is a local food delivery service in Amsterdam. Mobile application allows you to order food delivery from popular restaurants in Amsterdam.

Generalized functional requirements:
  • Order management
  • Contact with the deliverer
  • Location management
  • Discount management
  • Local data storage
  • Account management
  • Authorization
  • Registration
  • Password recovery
Technology stack on iOS client:
  • Swift
  • Storyboard
  • REST API
  • JWT
  • Core Data
  • Keychain
Technology stack on Android client:
  • Kotlin
  • Navigation
  • ViewModel
  • Retrofit
  • REST API
  • JWT
  • Room
  • SharedPreferences
  • Dagger2
Technology stack on server:
  • PHP
  • REST API
  • JWT
  • MySQL
